# LeetCode Assignment Solutions

Coleccion de soluciones de LeetCode organizada por dificultad.
El objetivo es tener un repo limpio, facil de revisar y listo para entrega academica.

## Estructura

- `easy/`: problemas nivel facil.
- `medium/`: problemas nivel intermedio.
- `hard/`: problemas nivel avanzado.
